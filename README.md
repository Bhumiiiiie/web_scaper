Book Data Scraper & Analyzer

This project contains two Python scripts:

1. scrape_books.py
This script scrapes book information (title, price, rating) from the website books.toscrape.com and saves it into a CSV file.
Features:

Fetches book titles
Extracts book price
Extracts book rating
Stores everything in books.csv

You can run it using:
python scrape_books.py

2. analyze_books.py
This script reads the books.csv file generated by the scraper and performs basic analysis on the dataset.
Features:

Shows highest priced book
Shows lowest priced book
Calculates average price
Counts number of books

Run it using:
python analyze_books.py
